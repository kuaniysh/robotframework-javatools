
If you are not behind proxy you can skip to step 3.

1) Configure your javaws proxy settings (run 'javaws -viewer' from command line).

2) Uncomment the suite setup and configure the "Set Proxy" user keyword to match your proxy settings.

3) From the command line run:

  On Windows:
  run.py robot-javaws-test.html

  On Linux:
  ./run.py robot-javaws-test.html
